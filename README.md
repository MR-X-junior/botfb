# botfb
Bot Facebook

# FEATURES
- [x] Bot Like post
- [x] Bot Unlike post
- [x] Bot Comments Post
- [x] Bot Share post
- [x] Bot Follow
- [x] Bot Unfollow
- [x] Dump Token
- [x] Get Token

## INSTALLATION
  
    $ pkg upgrade && pkg update

    $ pkg install python2

    $ pip2 install requests
	
    $ pip2 install bs4

    $ pkg install ruby
	
    $ pkg install git
	
    $ git clone https://github.com/MR-X-Junior/botfb
	
    $ cd botfb

## SETUP PROGRAM
    $ python2 setup.py

## RUN PROGRAM
    $ ruby run.rb
